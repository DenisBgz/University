# Wheel of Fortune

## Описание проекта

"Поле Чудес" - это игра, в которой игроки угадывают слово, загаданные компьютером, буква за буквой. Игра основана на классическом телевикторине и предназначена для развлечения и тренировки логического мышления.

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ваш_репозиторий/wheel_of_fortune.git
   ```

2. Перейдите в директорию проекта:
   ```bash
   cd wheel_of_fortune
   ```

## Запуск

Чтобы запустить игру, введите следующую команду в терминале:
```bash
python __main__.py
```

При успешном запуске вы сможете насладиться игрой и угадывать слова!


Чтобы собрать игру в исполняемый файл, выполните команду `bash build.sh`
